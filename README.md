# PC Тян 🤖💥

**PC Тян** – это программа, которая превращает ваш компьютер в аниме-девочку, реагирующую на хлопки и щелчки! Ударьте по столу, щёлкните пальцами или просто пошуршите – и ваша PC-тян ответит вам милым (или не очень) звуком из папки `moan`.

> ⚠ **Внимание**: проект создан исключительно в развлекательных целях. Не обижайте свою PC-тян слишком сильно!

## ✨ Особенности

- 🎤 **Реагирует на звуки**: хлопки, щелчки, постукивания – всё идёт в дело.
- 🎵 **Своя библиотека звуков**: просто положите любимые .wav или .mp3 в папку `moan`.
- 🔧 **Автоматическая настройка**: сам ищет ваш микрофон и подбирает параметры.
- 📈 **Адаптивный порог**: подстраивается под фоновый шум, чтобы не срабатывать на тишину.
- 🖥 **Работает на Windows**: благодаря магии `sounddevice` и `pygame`.

## 🚀 Установка

1. **Клонируйте репозиторий**  
   ```bash
   git clone https://github.com/yourusername/pc-tyan.git
   cd pc-tyan
   ```

2. **Установите зависимости**  
   ```bash
   pip install -r requirements.txt
   ```
   Или вручную:  
   ```bash
   pip install sounddevice numpy pygame
   ```

3. **Добавьте звуки**  
   Создайте папку `moan` в той же директории, что и скрипт, и положите туда любые `.wav` или `.mp3` файлы. Чем больше – тем веселее!

## 🎮 Использование

Запустите скрипт:

```bash
python pc_tyan.py
```

Теперь просто хлопайте, щёлкайте или стучите по столу – PC-тян будет отвечать случайным звуком из вашей коллекции. Для выхода нажмите `Ctrl+C`.

## ⚙️ Настройка

Вы можете подкрутить параметры в начале файла `pc_tyan.py`:

- `INITIAL_TAP_THRESHOLD` – чувствительность (чем меньше, тем тише звуки ловит).  
  *По умолчанию: `0.0002` – очень чувствительно.*
- `COOLDOWN_SECONDS` – задержка между воспроизведениями, чтобы не спамить.  
  *По умолчанию: `4` секунды.*
- `RATE` – частота дискретизации микрофона (должна совпадать с настройками в Windows).  
  *По умолчанию: `48000`.*

Если хотите увидеть реальные значения громкости (чтобы точно подобрать порог), раскомментируйте строку `# print(f"RMS: {amplitude:.6f}")` в функции `audio_callback`.

## 🐛 Возможные проблемы на Windows

### ❌ "Не удалось открыть микрофон"
- **Разрешения**: Зайдите в `Параметры → Конфиденциальность → Микрофон` и разрешите доступ приложениям (особенно Python).
- **Монопольный режим**: В `Панель управления → Звук → Запись` выберите микрофон, откройте `Свойства → Дополнительно` и снимите галочку "Разрешить монопольный режим".
- **Запуск от администратора**: Попробуйте запустить терминал от имени администратора.

### ❌ Звуки не проигрываются
- Убедитесь, что в папке `moan` есть файлы и они поддерживаются (`pygame` понимает WAV и MP3).
- Проверьте, не отключён ли звук в системе.

## 📦 Зависимости

- Python 3.7+
- `sounddevice` – захват звука с микрофона
- `numpy` – обработка аудиоданных
- `pygame` – воспроизведение звуков
- `ctypes` – инициализация COM (встроено)

Установить всё одной командой:
```bash
pip install sounddevice numpy pygame
```
